java windows下的配置环境变量

比如我的D:\java_jdk\bin 添加到path就好了。

也可以新建系统级别的变量： JAVA_HOME D:\java_jdk\bin 使用的时候用 %JAVA_HOME%

如何配置 临时 环境变量:

查看环境变量： set path 查看

修改环境变量： set path=hahah(会覆盖)

删除环境变量： set path=

新增环境变量： set path=%path%;E:\java_jdk\bin

javac 编译 记事本写的java文件时，如果存成utf8会报错。
